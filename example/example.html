Examples

Simple Path

vm.exampleSelect = ko.select(vm, "a.b.c");

Path to Array Index

vm.exampleSelect = ko.select(vm, "a.b.c[1]");

Path to negative Array Index

vm.exampleSelect = ko.select(vm, "a.b.c[-1]");

Filter an array

vm.exampleSelect = ko.select(vm, "project.users{name=dennis}");

Filter an array, based on sub properties

vm.exampleSelect = ko.select(vm, "project.documents{owner.name=dennis}");
